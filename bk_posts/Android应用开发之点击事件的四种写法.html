Android应用开发之点击事件的四种写法

Android应用开发之点击事件的四种写法

第一种方法

●先在布局里面增加一个按钮控件

<Button
    android:id="@+id/btn1"
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:layout_centerHorizontal="true"
    android:layout_centerVertical="true"
    android:text="点击" />

注意:上面按钮控件的id中，@+id表示创建一个控件，@id表示引用一个控件

●然后在MainActive.java中的onCreate方法中操作

1、获取控件

Button botton = (Button) findViewById(R.id.btn1);

findViewByid()返回的是一个控件，但是我希望使用按钮控件来定义，本来按钮这个类就是控件的子类，这里注意要强制转换一下，传入参数是这个控件的id，值得一提的是很多控件的id都在R这个类的id类下。

2、设置控件的点击事件

botton.setOnClickListener(new OnClickListener(){

        @Override
        public void onClick(View arg0) {
            //点击事件的内容
        }

    });

直接调用Button的方法即可，其参数是一个接口类型，在本例子中选择的是new一个局部内部类，名称就是这个接口的名称，然后覆盖掉里面的方法

3、设置点击事件的内容

public void onClick(View arg0) {

    System.out.println("zhuyanlin");

}

比如上面的System.out.println()语句就是一个内容，在这里的操作将会在点击事件触发时生效，即按下按钮的时候。

第二种方法

●增加按钮控件方式和第一种方法一致

●除了设置控件点击事件，其他与第一种方法一致

2、设置控件的点击事件

button.setOnClickListener(this);

setOnClickListener()的参数让当前类去实现，即MainActivity类去实现

public class MainActivity extends Activity implements OnClickListener

让MainActivity类去引用OnClickListener这个借口，并在当前类中覆盖方法

@Override
public void onClick(View arg0) {

    System.out.println("zhuyanlin");

}

在此方法中编写点击事件的执行代码即可。

第三种方法

●增加按钮控件方式和第一种方法一致

●除了设置控件点击事件，其他与第一种方法一致

2、设置控件的点击事件

button.setOnClickListener(new MyClickListener());

使用一个定义的类去实现这个接口

class MyClickListener implements OnClickListener {

    @Override
    public void onClick(View arg0) {
        System.out.println("zhuyanlin");

    }
}

可在MainActivity类中自定义一个内部类，并且引入这个接口即可。

第四种方法

●先在布局里面增加一个按钮控件

<Button
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:layout_centerHorizontal="true"
    android:layout_centerVertical="true"
    android:onClick="MyClick"
    android:text="点击" />

这里把按钮的id给删除掉了，然后添加了android:onClick=”MyClick”这行代码，其中MyClick是一个方法名。

●然后在MainActive.java中的onCreate方法中操作

public void MyClick(View v) {
    System.out.println("zhuyanlin");
}

在MainActivity类中定义一个形如上述的方法即可，一定要public void的形式，并且参数一定是一个View形式的参数，然后在其中编写点击事件的代码即可。

注意：虽然第四种方法比较简单，代码也比较少，但是不推荐使用这种方法，连id都没有，而且比较跳跃，不像第一种方法，第一种方法比较直白，所以推荐第一种方法。
